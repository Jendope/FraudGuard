How It Works

RAG Pipeline Architecture

A four-stage pipeline combining retrieval and generation for accurate fraud detection.

01

Data Collection

Web scraping pipeline harvests fraud cases from HK01 news API. 606 articles collected from Jan 2024 - Feb 2026.

02

Vector Embedding

Articles are embedded using Sentence-BERT (text2vec-base-chinese) and stored in ChromaDB vector database.

03

Semantic Retrieval

When you submit a query, we retrieve the top 3 most similar fraud cases using cosine similarity.

04

LLM Analysis

DeepSeek-v3.2 analyzes the query with retrieved context, producing a fraud score (0-10) with explanation.

# Data Collection

> Web scraping notebooks, daily update scripts, and collected fraud article datasets.

## Contents

| File | Description |
|------|-------------|
| `main.ipynb` | Primary data collection notebook |
| `hk01-web-scraping.ipynb` | HK01 web scraper — fetches fraud articles from HK01 API |
| `daily-update.ipynb` | Incremental daily update script for the knowledge base |
| `adcc_scam_articles.md` | Anti-Deception Coordination Centre (ADCC) scam articles |
| `hk01_scam_articles.md` | HK01 news fraud articles |

## Maintainers

This folder is maintained by the data collection team. Notebooks are used to scrape and curate fraud case datasets from publicly available Hong Kong news sources.

## Usage

Open `main.ipynb` in Jupyter Notebook to run the data collection pipeline:

```bash
cd data-collection
jupyter notebook
```
